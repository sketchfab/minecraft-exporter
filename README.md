** Windows
install:
* python2.7
* pyqt4
* easy_install https://pypi.python.org/pypi/setuptools
* easy_install pip
* easy_install poster
* pip install urllib2

** OSX
install:
* setuptools
* py2app
* pyqtx http://sourceforge.net/projects/pyqtx/
* run the script build.sh to create a standalone application
